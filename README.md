# InPrOp-Instagram-Profile-Opener
Chrome Browser extension to automatically visit provided Instagram profiles with short delay (3 seconds) between entries (created with the help of ChatGPT).

Profiles need to be provided as either a link to an instagram profile or as the unique username into a text field with newlines between each entry. Sometimes the first entry is skipped and needs to be redone.
